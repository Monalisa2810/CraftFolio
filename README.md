# 🚀 CraftFolio AI

## 🌟 Overview

CraftFolio AI is an AI-powered Resume Builder and Portfolio Generator designed to help students, professionals, and job seekers create industry-ready resumes and personal portfolio websites with minimal effort.

The platform combines Resume Creation, ATS Optimization, AI-powered Content Generation, and Portfolio Website Generation into a single seamless workflow.

---

## 🔗 Live Demo

Website:

https://craftfolio-liz.onrender.com/

## 🎯 Problem Statement

Many students and job seekers struggle with:

* Creating professional resumes
* Writing impactful career summaries
* Optimizing resumes for ATS systems
* Building personal portfolio websites
* Presenting their skills effectively online

CraftFolio AI solves these challenges by leveraging Artificial Intelligence and modern web technologies.

## ✨ Key Features

### 📝 Resume Builder

Create professional resumes through an intuitive user interface.

### 🤖 AI Summary Generator

Generate professional summaries and career objectives using Gemini AI.

### 📊 ATS Resume Analysis

Evaluate resume quality and receive ATS optimization suggestions.

### 🎨 Multiple Resume Templates

Choose from professionally designed resume layouts.

### 📄 PDF Export

Download resumes in PDF format.

### 🌐 AI Portfolio Generator

Convert resume information into a portfolio website automatically.

### 🔐 User Authentication

Secure login and registration using JWT authentication.

### ☁️ Cloud Storage

Store and manage resume information using MongoDB Atlas.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Monalisa2810/CraftFolio.git
```

Install frontend dependencies:

```bash
cd frontend
npm install
```

Install backend dependencies:

```bash
cd backend
npm install
```

Run backend:

```bash
npm start
```

Run frontend:

```bash
npm run dev
```
